# 接口：addCourse  [返回](../README.md)
用例： [添加课程](../用例/添加课程.md)
- 功能：
    添加课程。
    
- 权限：
    老师：只有老师可以调用该API。    
    
- API请求地址： 
    接口基本地址/v1/api/addCourse

- 请求方式 ：
    POST

- 请求实例：

        {
            "name":"信息系统分析与设计",
        }
        
- 请求参数说明:        

  |参数名称|说明|
  |:---------:|:--------------------------------------------------------|      
  |name|课程名| 
  
- 返回实例：

        { 
            "status": true,
            "info": null,    
        }
 
- 返回参数说明：    
 
  |参数名称|说明|
  |:---------:|:--------------------------------------------------------|      
  |status|bool类型，true表示正确的返回，false表示有错误|
  |info|返回结果说明信息|
